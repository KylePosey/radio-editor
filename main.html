h4>Welcome to team Lucid's Radio Editor Tool!
It is highly recomended you use this tool on desktop.
Please read the help box for information on using the tool and if you need help on replacing the file be sure to read the help box on the radiopanel.txt window.

This tool allows you to put custom chat binds into the radio menu, either by replacing commands you dont use or by taking advantage of the 3/4 remaining slots of a radio menu. This tool allows you to edit radio1, radio2, and radio3. I purposely left out the ability to edit radio (radio0) since that radio contains all bot commands and a yes/no voice command.

Click download to download RadioPanel.txt, or click inside the panel to select all text to copy over.
Input desired radio commands into the radio menu below
Download and replace file OR copy and paste to the default file. Filepath:
C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\resource\ui\radiopanel.txt
Inside of console, bind the radio that was created with this tool. Example: bind [KEY] radio3
